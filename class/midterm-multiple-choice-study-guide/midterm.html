HTML

What is the difference between your <head> and <body> tags?

This is an answer The head tag stores meta data and unseen information for the page and the body tag stores what is visible on the page.

What are the main tags that go in the <head>?

  • Title- What is displayed on the tab of the webpage.
  • Meta- Tags that store meta data about the page for search engines to read, important for making pages responsive.
  • Link- linking css and other pages.

What is the <meta> viewport tag for? Why is it important?

Defines the screen size and tells the browser to be responsive based on screen size.

How do you link a CSS file in your HTML?

<link rel=stylesheet href="style.css" type="text/css"> Using a link tag

What are the main tags that go in the <body>?

  • Heading tags (h1-h6)
  • (p) Paragraph tags
  • div tag
  • Span
  • (ul) unordered list / (ol) ordered list / (li) list item
  • image (img)
  • link (a)
  • header
  • footer

How do you add an image to your HTML?

< img source="folder/filename" alt="text for when image wont display" >

What are the three different <a> types in HTML?

  • internal link to another html page
  • external link to another site
  • anchor link to jump down the page

What property is used on the <a> tag to open a link in a new tab? Write out a link to Google using the property to open in a new tab.

What are <div> and <span> tags? How are they used?

< div > tags are block elements used as containers and wrappers for targeting elements. < span > tags are used to target inline elements.

How do you add a border to an image? What about margins?

                img {
                property: value;
                border: 1px black solid;
                border: [width] [style] [color]
                }

What are the three ways you can format padding (and margin) values when defining different sides of an element?

                img {
                padding: 10px; // applies padding to all sides
                padding: 5px 10px; // top/ bottom / top/bottom then left/right
                padding: top right bottom left; // defines sides
                }

What does `margin: 0 auto;` mean with reference to the question above?

it means no margin on the top and bottom and auto margins on the left and right, centers content. Used to center content on a page. Adds same ammount of space to sides of element if its width is defined.

What is a hexadecimal code? How is it used?

used to define colors in css, #000000

What is a media query? Why is it used? Write a media query that changes the background color of your html page on small screen sizes.

makes a page responsive, changes the styles of a page for the different defined screen sizes (min-width) (max-width) 

                
                @media screen and (max-width: 500px) {
                    body {
                        property: value;
                    }
                }

What is flexbox? Why is it used?

A set of css proterties and values used to define the position of elements on a page within a container. Useful for layouts and alignment.

Formatting and validating

What is the value of formatting your code?

Readibility, clarity, easier for others to understand. See whats nested. Find mistakes/ debug.

Why is it especially important to have clean, readable CSS code?

Order matters. If something is defined 2x it will go with whichever you defined later. Specific selectors make it easier to not override your own code.

What is a validator? Why do you use it?

It reads code and checks for any issues. Helps with debugging and working through errors.
